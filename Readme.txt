== Description ==

Make Styled Note box, Alert box, Help box, Tip Box, Important Box in your posts using classes.

== Installation ==

To install the plugin follow these steps :

1. Download the style-box.zip file to your local machine.
2. Unzip the file 
3. Upload "style box" folder to the "/wp-content/plugins/" directory
4. Activate the plugin through the 'Plugins' menu in WordPress

That's it.
